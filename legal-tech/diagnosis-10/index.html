### ケース別記入例

**例A：フリーランスのWeb制作代金（33万円）**

| 項目 | 記入内容 |
|------|----------|
| 契約内容 | Webサイト制作業務（トップページ＋下層5ページ） |
| 契約日 | 令和8年1月15日 |
| 納品日 | 令和8年2月28日 |
| 請求金額 | 金330,000円（税込） |
| 支払期限 | 令和8年3月31日 |

**例B：個人間の売買代金（8万円）**

| 項目 | 記入内容 |
|------|----------|
| 契約内容 | 中古カメラ（型番○○）の売買 |
| 契約日 | 令和8年2月1日 |
| 納品日 | 令和8年2月3日（配送完了日） |
| 請求金額 | 金80,000円 |
| 支払期限 | 令和8年2月10日 |

### 送付方法

1. **郵便局の窓口**で「内容証明郵便（配達証明付き）」と伝える
2. 同じ文面を**3通**用意（相手方・自分・郵便局の控え）
3. 料金：基本郵便料 110円 ＋ 内容証明加算 480円（1枚）＋ 配達証明 350円 ＝ **約1,300〜1,800円**
4. 書式制限：1行20字×26行（縦書き）または1行26字×20行（横書き）
5. e内容証明（ネット発送）も利用可：https://e-naiyo.post.japanpost.jp/

### よくある失敗例

- **金額の計算ミス**：消費税込み・込みなしが契約書と一致しない → 請求書・見積書と照合すること
- **相手の名前・住所の誤記**：法人名が略称のまま → 登記簿上の正式名称を使う
- **期限が短すぎる**：「3日以内」は不当に短い → **14日程度**が一般的
- **感情的な文面**：「詐欺だ」「訴えてやる」等の過激な表現 → 事実のみ淡々と記載する

## テンプレート2：クーリングオフ通知書

### 法的根拠

- **特定商取引法第9条**（訪問販売のクーリングオフ：8日間）
- **特定商取引法第24条**（電話勧誘販売のクーリングオフ：8日間）
- **特定商取引法第40条**（連鎖販売取引：20日間）
- **特定商取引法第48条**（業務提供誘引販売：20日間）

### テンプレート本文

        記

  2. 商品名（役務名）：[商品名・サービス名]

### ケース別記入例

**例A：訪問販売の浄水器（25万円・契約から5日目）**

| 項目 | 記入内容 |
|------|----------|
| 法律条文 | 第9条（訪問販売） |
| 契約日 | 令和8年3月1日 |
| 商品名 | 家庭用浄水器「○○クリーン」一式 |
| 契約金額 | 金250,000円 |
| 支払済み金額 | 金250,000円（クレジットカード一括） |

**例B：電話勧誘の教材セット（18万円・契約から3日目）**

| 項目 | 記入内容 |
|------|----------|
| 法律条文 | 第24条（電話勧誘販売） |
| 契約日 | 令和8年2月20日 |
| 商品名 | 英会話教材「○○マスター」DVD全12巻セット |
| 契約金額 | 金180,000円 |
| 支払済み金額 | 金0円（初回引落前） |

### 送付方法

1. **はがきでも有効**だが、証拠保全のため**特定記録郵便**または**簡易書留**を推奨
2. 発送前にはがき・書面の**両面をコピーまたは写真撮影**して保管
3. **発信主義**：期間内に発送すればOK（届いた日ではない）
4. クレジット契約がある場合、**クレジット会社にも同時に通知**する
5. 料金：特定記録郵便 244円 ＋ 基本郵便料 110円 ＝ **約354円**

### よくある失敗例

- **期間の数え間違い**：契約書面を受け取った日が1日目 → 受領日を必ず確認
- **「クーリングオフできない」と言われて諦める**：業者の虚偽説明の場合は期間延長される（法第9条第1項ただし書）
- **電話だけで済ませる**：口頭では証拠が残らない → 必ず**書面で通知**
- **商品を使ってしまう**：未開封でも開封済みでも権利は消えないが、返品時のトラブルを避けるため未使用が望ましい

## テンプレート3：敷金返還請求書

### 法的根拠

- **民法第621条**（賃借人の原状回復義務：通常損耗は含まない）
- **民法第622条の2**（敷金の返還義務）
- **国土交通省「原状回復をめぐるトラブルとガイドライン」**（再改訂版）
