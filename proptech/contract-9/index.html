## ステップ3：割引率の決定と支払スケジュール作成

### 目的
リース負債の現在価値計算に必要な割引率を決定し、支払スケジュールを確定する。

### チェックリスト

| # | タスク | 抽出元 | 担当 | 期限目安 | 完了 |
|---|--------|--------|------|----------|------|
| 3-1 | 契約ごとの計算利子率を確認（明示の場合） | 契約書・覚書 | [記入例: 経理部 高橋] | [記入例: 第5週] | ☐ |
| 3-2 | 追加借入利子率（IBR）を算定 | 取引銀行見積・社債利回り | [記入例: 財務部 中村] | [記入例: 第5週] | ☐ |
| 3-3 | 適用割引率を契約ごとに決定 | 3-1,3-2の結果 | [記入例: 経理部 部長承認] | [記入例: 第6週] | ☐ |
| 3-4 | 月次支払スケジュールを作成 | 賃料・改定条件 | [記入例: 経理部 高橋] | [記入例: 第6週] | ☐ |
| 3-5 | 監査法人へ割引率の事前相談 | 算定根拠資料 | [記入例: 経理部 部長] | [記入例: 第7週] | ☐ |

### CSV列定義（ステップ3出力）

| 列名 | 型 | 記入例 |
|------|----|--------|
| 適用割引率_% | DECIMAL | `2.5` |
| 割引率根拠 | TEXT | `IBR（みずほ銀行5年固定）` |
| 支払回数 | INT | `60` |
| 支払頻度 | TEXT | `月次` |
| 支払月額_1期 | INT | `1800000` |
| 支払月額_改定後 | INT | `1854000` |
| 改定適用開始月 | INT | `25` |
| 年間支払総額 | INT | `21600000` |
| 支払スケジュールJSON | TEXT | `[{"month":1,"amount":1800000},...]` |

## CSV連携ワークフロー

商業リース条項 横断比較ツール │ │ CSV出力（12条項＋会計項目） ▼ ステップ1-2 CSV ─→ 条項抽出・構造化完了 │ │ 割引率・スケジュール追加 ▼ ステップ3 CSV ─→ 財務部レビュー │ │ 現在価値計算 ▼ ステップ4 CSV ─→ 監査法人レビュー │ │ 仕訳・開示追加 ▼ ステップ5 最終CSV ─→ 会計システム取込み
