重説で見落とすと100万円損する5つのコスト項目

マンションや戸建ての購入を決め、不動産会社から重要事項説明書（重説）を受け取ったとき、あなたは何ページ目で集中力が切れるだろうか。

重説は数十ページに及ぶ法律文書だ。しかし、その中には購入後5年・10年で数十万〜100万円超の追加コストに直結する条項が埋まっている。本記事では、実際に見落とされやすい上位5項目を、具体的な金額シナリオ付きで解説する。

記事内の質問番号は、連動チェックリスト「重説を受け取ったら聞く「お金の質問」20選」に対応しています。帰宅後は PropCostScan｜重説コストスキャナー に数値を入力し、追加コストを可視化しましょう。

コスト1｜修繕積立金の段階値上げ — 10年で+200万円も

関連質問: 質問1 ⚠️・質問2 ⚠️・質問3

どこに書いてある？

重説の「管理費・修繕積立金」欄。現在の月額は記載されるが、将来の値上げスケジュールは長期修繕計画書を見なければわからない。

具体的な金額シナリオ

時期 月額（値上げなし想定） 月額（実際の計画） 差額（月）
購入時 8,500円 8,500円 0円
築10年（5年後） 8,500円 12,800円 +4,300円
築15年（10年後） 8,500円 15,200円 +6,700円
  • 5年間の追加コスト: 4,300円 × 12か月 × 約3年（値上げ後期間）＝ 約15.5万円
  • 10年間の追加コスト: 段階値上げを累計すると 約60〜80万円

見落とすとこうなる

「月8,500円なら安い」と思って購入したのに、5年後に月額が1.5倍に。さらに修繕積立金の残高不足で一時金30〜50万円を徴収されるケースもある。合計で100万円超の想定外出費になり得る。

防ぎ方

  • 質問1で長期修繕計画書を必ず見せてもらう
  • 質問2で積立金の現在残高と直近の大規模修繕時期を確認する
  • PropCostScanに現在の月額と計画書の値上げ時期を入力すれば、5年・10年の累計コストが一目でわかる

コスト2｜固定資産税の軽減終了 — 税額が突然2倍に

関連質問: 質問7・質問8

どこに書いてある？

重説の「公租公課」欄に年額が記載される。ただし軽減措置の適用中かどうかは、自分から聞かないと教えてもらえないことが多い。

具体的な金額シナリオ

期間 固定資産税（年額） 備考
購入〜軽減終了まで 98,000円 新築軽減（1/2）適用中
軽減終了後 185,000円 本来の税額に戻る
  • 新築マンションの軽減期間は5年、戸建ては3年
  • 軽減終了後の5年間の追加負担: (185,000 − 98,000) × 5 ＝ 約43.5万円

見落とすとこうなる

「年10万円弱なら問題ない」と資金計画を立てたのに、数年後に税額がほぼ倍増。住宅ローン返済に加え、年間約9万円の追加負担が重くのしかかる。

さらに、不動産取得税（質問8）は入居後半年〜1年で届く。軽減申告をしないと数十万円が満額請求される。例えば課税標準額1,800万円の物件なら、軽減なしで54万円、軽減申告すれば0円になるケースもある。

防ぎ方

  • 質問7で軽減措置の残り期間と終了後の税額を具体的に聞く
  • 質問8で不動産取得税の概算と軽減申告の要否を確認する
  • PropCostScanの「税金・公租公課」セクションに入力すれば、軽減終了後を含む10年間の税負担を自動計算できる

コスト3｜手付解除・違約金の二重構造 — 最大で売買価格の20%

関連質問: 質問4 ⚠️・質問5・質問6

どこに書いてある？

重説の「契約の解除に関する事項」欄。手付金の額と、違約金の率が別々に記載されている。

具体的な金額シナリオ（売買価格4,000万円の場合）

解除パターン 負担額
手付解除（手付金放棄のみ） 100万円
契約違反による解除（違約金20%） 800万円
ローン特約期限切れ後の解除 手付100万円 + 違約金800万円 ＝ 900万円
ローン特約期限内の解除 0円（白紙解除）

見落とすとこうなる

「手付金100万円を捨てれば解約できる」と思い込んでいたが、実は違約金条項が別途あり、売買価格の20%＝800万円を請求された——という事例は実在する。

特に危険なのがローン特約の期限切れ（質問5）。期限内なら0円で白紙解除できるのに、1日でも過ぎると手付放棄＋違約金の二重負担になる。

防ぎ方

  • 質問4で「手付放棄とは別に違約金条項があるか」を明確に確認する
  • 質問5でローン特約の期限日をカレンダーに登録する
  • 賃貸の場合は質問6で中途解約違約金を確認（1年未満の解約で家賃1か月分が一般的）

コスト4｜都市計画制限による建替え・改修コスト増

関連質問: 質問9・質問10・質問11

どこに書いてある？

重説の「都市計画法・建築基準法に基づく制限」欄。用途地域、建ぺい率、容積率、高度地区制限などが記載される。

具体的な金額シナリオ

リスク 追加コスト目安
既存不適格で建替え時に延床面積20%減 資産価値 −300〜500万円
セットバック義務（前面道路4m未満） 土地面積の減少＋外構工事 50〜150万円
用途地域変更で改修に追加申請・設計変更 30〜80万円
都市計画道路の計画線にかかる 将来の立退き・補償交渉リスク

見落とすとこうなる

「駅近で安い」と飛びついた物件が、実は建ぺい率オーバーの既存不適格だった。今の建物に住む分には問題ないが、将来建替えると今より20%小さい家しか建てられない。売却時にも買い手がつきにくく、資産価値が数百万円下がる。

防ぎ方

  • 質問9で「増築・建替え時に現在より小さくなるか」を直接聞く
  • 質問10でセットバック義務の有無を確認する
  • 質問11で都市計画道路や再開発計画の予定を確認する
  • PropCostScanでは建築制限リスクを「低・中・高」で判定し、該当する場合の概算コストを表示する

コスト5｜地盤・浸水リスクによる保険料増と修繕費

関連質問: 質問12・質問13・質問14

どこに書いてある？

重説の「土砂災害警戒区域」「浸水想定区域」「造成宅地防災区域」欄。ハザードマップとの照合結果が記載される。

具体的な金額シナリオ

リスク要因 追加コスト（10年間）
浸水想定区域で水災補償付き火災保険 保険料 +20〜40万円
地盤が弱いエリアでの地震保険割増 保険料 +10〜25万円
浸水被害後の修繕（床上30cm想定） 1回あたり 100〜300万円
地盤沈下による基礎補修 50〜200万円

見落とすとこうなる

重説に「浸水想定区域：該当あり（想定浸水深0.5〜3.0m）」と小さく書かれていたのを読み飛ばした。入居3年後に豪雨で床上浸水し、修繕費150万円と仮住まい費用50万円が発生——保険の水災補償にも入っていなかった。

防ぎ方

  • 質問12でハザードマップ上のリスク区分を具体的に確認する
  • 質問13で過去の浸水・地盤沈下の履歴を聞く
  • 質問14で地盤調査データの有無を確認する
  • PropCostScanに地域のハザード情報を入力すると、保険料の増加額と被災時の想定コストを算出できる

まとめ

重説の「隠れコスト」は、知っていれば防げるものばかりだ。

  • 修繕積立金 → 長期修繕計画書を見れば将来の月額がわかる
  • 固定資産税 → 軽減期間の残りを聞けば終了後の税額を予測できる
  • 違約金 → 手付金とは別の条項がないか確認すれば二重負担を回避できる
  • 都市計画制限 → 建替え可能な面積を聞けば資産価値の毀損を判断できる
  • 地盤・浸水リスク → ハザード情報を確認すれば保険設計に反映できる
