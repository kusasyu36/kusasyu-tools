重要事項説明書 買主セルフリスク診断ツール「PropShield」

重説「ここだけは見て」30秒チェックシート

使い方: 重要事項説明書を受け取ったら、以下5項目を上から順にチェック。各項目に✅または❌を記入し、1つでも❌があれば契約前に必ず確認・交渉してください。所要時間：約30秒。

✍️ 物件情報（記入欄）

項目 内容
物件名・所在地 [記入例: 東京都世田谷区○○1-2-3 ○○マンション301号室]
確認日 [記入例: 2026年3月9日]
確認者 [記入例: 山田太郎]
判定結果 全項目OK → 契約手続きへ / NG有り → 下記アクション実施

チェック① 瑕疵担保責任（契約不適合責任）の範囲

項目 内容
確認箇所 重説「瑕疵担保責任（契約不適合責任）」欄
判定 ✅ or ❌：______

OK基準 - 売主が宅建業者→ 引渡しから2年以上の保証期間がある - 売主が個人→ 保証期間が3ヶ月以上ある（一般的には3ヶ月〜1年） - 対象範囲に「構造耐力上主要な部分」「雨水の浸入を防止する部分」が含まれている

NG基準 - 「瑕疵担保責任を負わない」「免責」と記載されている - 保証期間が極端に短い（業者売主で2年未満） - 対象範囲が限定的すぎる（「設備のみ」など）

NGの場合のアクション 1. 売主に保証期間の延長・免責撤回を交渉 2. 交渉不可の場合、ホームインスペクション（住宅診断） を契約前に実施 3. 既存住宅売買瑕疵保険の付保を検討

よくある落とし穴

個人間売買で「現状有姿・免責」が慣例とされがちだが、交渉で条件を付けることは可能。「免責だから仕方ない」と鵜呑みにしない。

チェック② ハザードマップ該当有無

項目 内容
確認箇所 重説「水害ハザードマップ」「土砂災害警戒区域」欄
判定 ✅ or ❌：______

OK基準 - 洪水浸水想定区域：区域外、または浸水想定0.5m未満 - 土砂災害警戒区域：非該当 - 津波災害警戒区域：非該当

NG基準 - 浸水想定3m以上（2階床上浸水レベル） - 土砂災害特別警戒区域（レッドゾーン）に該当 - 複数のハザードに重複該当

NGの場合のアクション 1. 自治体のハザードマップポータルで最新版を自分でも確認 2. 火災保険の水災補償の保険料を事前見積もり（年数万円の差になる場合あり） 3. 浸水想定3m以上の場合は購入再検討を強く推奨

よくある落とし穴

重説に添付されるハザードマップは縮小・白黒コピーで見づらいことが多い。必ず自治体HP等でカラー版の原本を確認すること。2020年の法改正で説明義務化されたが、「該当なし」の誤記載事例も報告されている。

チェック③ 接道義務の充足

項目 内容
確認箇所 重説「道路との関係」「接道」欄
判定 ✅ or ❌：______

OK基準 - 建築基準法上の道路（幅員4m以上）に2m以上接している - 道路の種別が「第42条1項◯号道路」と明記されている

NG基準 - 接道が2m未満（＝再建築不可の可能性大） - 道路幅員が4m未満でセットバックが必要 - 「建築基準法上の道路に該当しない」旨の記載がある

NGの場合のアクション 1. 接道2m未満 → 再建築不可物件の可能性。住宅ローンが通らないケースが多い。購入を慎重に再検討 2. セットバック要 → 有効敷地面積が減少。実質の建築可能面積を再計算 3. 市区町村の建築指導課で建築確認の可否を事前確認

よくある落とし穴

「道路に面している」と「建築基準法上の接道義務を満たしている」は別。見た目は道路でも法律上は「通路」扱いの場合がある。旗竿地（路地状敷地）は間口2mギリギリのケースが多く、実測で2m未満と判明する事例も。

チェック④ 管理費・修繕積立金の滞納額（マンションの場合）

項目 内容
確認箇所 重説「管理費等の滞納」「修繕積立金の積立状況」欄
判定 ✅ or ❌：______（※戸建ての場合は「該当なし」と記入）

OK基準 - 売主（当該住戸）の滞納額が0円 - 管理組合全体の滞納率が記載されていて深刻でない（目安：滞納率5%以下） - 修繕積立金の残高が長期修繕計画と整合している

NG基準 - 当該住戸に滞納がある（買主が承継する可能性あり） - 大規模修繕の予定があるのに積立金が不足している - 近年の管理費値上げ履歴がある（今後も上がる可能性）

NGの場合のアクション 1. 滞納ありの場合 → 売主に決済前の完済を条件に入れる（売買契約書に明記） 2. 積立金不足 → 長期修繕計画書と直近の総会議事録を取り寄せて確認 3. 将来の値上げリスクを月額ランニングコストに上乗せして資金計画を見直す

よくある落とし穴

滞納は区分所有法により買主に承継される（第8条）。「売主の問題だから関係ない」は誤り。また、修繕積立金が「異常に安い」物件は、将来の一時金徴収や大幅値上げのリスクが高い。月額㎡あたり200円未満は要注意。

チェック⑤ 建ぺい率・容積率の超過有無

項目 内容
確認箇所 重説「用途地域」「建ぺい率・容積率」欄、および建築確認通知書
判定 ✅ or ❌：______

OK基準 - 指定建ぺい率・容積率の範囲以内で建築されている - 建築確認済証・検査済証が存在する - 増改築の履歴がある場合、確認申請済みである

NG基準 - 既存建物が建ぺい率・容積率を超過している（＝既存不適格or違法建築） - 検査済証がない（完了検査を受けていない） - 増改築が無届けで行われた形跡がある

NGの場合のアクション 1. 既存不適格（建築時は適法→法改正で超過） → 住宅ローン審査が厳しくなる。融資可能な金融機関を事前確認 2. 違法建築（建築時から超過） → 住宅ローンがほぼ不可。購入を再検討 3. 検査済証なし → 台帳記載事項証明書を市区町村で取得し、建築確認の有無だけでも確認

よくある落とし穴

中古戸建ての約半数は検査済証がないとも言われる。検査済証がない＝即違法ではないが、ローン審査でのハードルは高い。また、カーポートや増築部分が建ぺい率超過の原因になっている「隠れ違反」は非常に多い。

総合判定

チェック項目 判定
①瑕疵担保責任 ✅ / ❌
②ハザードマップ ✅ / ❌
③接道義務 ✅ / ❌
④管理費等の滞納 ✅ / ❌
⑤建ぺい率・容積率 ✅ / ❌
総合 全項目✅ → 契約手続きへ / ❌あり → 要対応

❌が1つでもある場合： 契約予定日の最低1週間前までに不動産会社・売主に確認し、解決できない場合は契約延期または購入中止を検討してください。
