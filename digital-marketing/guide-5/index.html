### 手順2：トーン定義を貼り付け

1. 本テンプレートのセクション①を全選択してコピー
2. ツールの「ブランドトーン」テキストエリアに貼り付け
3. 基準レベル（1〜5）をプルダウンで選択

[画面イメージ] ┌─────────────────────────────────────┐ │ ブランドトーン設定 │ │ ┌─────────────────────────────┐ │ │ │ ここにセクション①を貼り付け │ │ │ │ │ │ │ └─────────────────────────────┘ │ │ 基準レベル: [▼ レベル3 ] │ └─────────────────────────────────────┘


### 手順3：NGワードリストを貼り付け

1. セクション②のうち、自社に該当する法律のNGワード表をコピー
2. 「NGワード」テキストエリアに貼り付け
3. 複数の法律に該当する場合はすべて貼り付けてOK

[画面イメージ] ┌─────────────────────────────────────┐ │ NGワード設定 │ │ 業界: [▼ 化粧品・健康食品 ] │ │ ┌─────────────────────────────┐ │ │ │ ここにセクション②を貼り付け │ │ │ │ │ │ │ └─────────────────────────────┘ │ │ ☑ 薬機法 ☑ 景表法 ☐ 金商法 │ └─────────────────────────────────────┘


### 手順4：推奨表現・差別化キーワードを貼り付け

1. セクション③の推奨表現テーブルと差別化キーワードテーブルをコピー
2. 「推奨表現」テキストエリアに貼り付け

### 手順5：設定を保存して監査テスト

1. 「設定を保存」ボタンをクリック
2. 「監査」タブに戻り、テスト用のAIコンテンツを貼り付け
3. 「監査を実行」ボタンで3つのスコアが表示されることを確認
   - **トーン一致度スコア**（0〜100点）
   - **法規制リスクスコア**（NGワード検出数と該当法律）
   - **競合差別化スコア**（0〜100点）
