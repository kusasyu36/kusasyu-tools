海外展開ローカライゼーション 発注ブリーフ・テンプレート

使い方

  1. 案件ごとにこのテンプレートをコピーし、各セクションの [ ] 内を記入してください
  2. 市場（米国・中国・韓国）ごとにセクション②〜⑤を個別に作成します。1市場のみの場合は該当市場だけ記入してください
  3. 記入後、翻訳会社・ネイティブレビュアーにこのブリーフごと共有してください
  4. 品質確認時はセクション⑤のスコアカードで70点以上を合格ラインとします
  5. 本ブリーフは「文化適応診断チェッカー｜市場別AI分析ツール」の診断結果と併用することで、修正指示の精度が向上します

⑤ 品質チェック基準（文化適応スコアカード）

納品物を以下の基準で採点してください。各項目10点満点×10項目＝100点満点。70点以上で合格。

# チェック項目 評価基準 点数（/10）
1 言語の自然さ ネイティブが読んで違和感がないか [ /10 ]
2 ブランドトーン一致 セクション①で定義したトーンに合っているか [ /10 ]
3 禁止表現の排除 セクション②のNG表現が含まれていないか [ /10 ]
4 文化的地雷の回避 現地で炎上・誤解を招く表現がないか [ /10 ]
5 プラットフォーム適合 セクション③の形式要件を満たしているか [ /10 ]
6 CTA の有効性 現地ユーザーが自然に行動したくなるCTAか [ /10 ]
7 ハッシュタグの適切さ 現地トレンド・検索ボリュームに合っているか [ /10 ]
8 競合水準との比較 セクション④の競合と同等以上の品質か [ /10 ]
9 法規制の遵守 広告法・薬機法相当の現地規制に違反していないか [ /10 ]
10 原文の意図保持 翻訳で失われがちなブランドメッセージが残っているか [ /10 ]

■ 補足：「文化適応診断チェッカー」との連携

本ブリーフ記入前に「文化適応診断チェッカー｜市場別AI分析ツール」で原文を診断し、検出された文化的リスク（トーン不適合・禁止表現・CTA表現の差異）をセクション②〜⑤に反映させると、発注精度が大幅に向上します。

推奨ワークフロー: 1. 原文を「文化適応診断チェッカー」に入力 → 市場別レポートを取得 2. レポートの検出項目を本ブリーフのセクション②（禁止表現）・⑤（スコアカード）に転記 3. 記入済みブリーフを翻訳会社・レビュアーに送付 4. 納品物をスコアカードで採点 → 70点未満なら修正依頼
