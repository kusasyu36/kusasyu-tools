BtoBコンテンツ戦略マップ管理シート

商談につながるBtoBコンテンツ自動生成ツール

    A列         B列      C列       D列       E列       F列

1行 [空白] 記事 資料 動画 事例 比較表 2行 認知 (入力↓) 3行 比較 4行 決定


各交差セル（B2〜F4）の中に、**3つの情報**を縦に記入します。

- **URL**（例: B2） — コンテンツのURL
- **月間PV**（例: B5） — 直近30日のページビュー
- **月間CV数**（例: B8） — 資料DL・問い合わせ等のコンバージョン数

実際のレイアウトでは行を拡張し、以下のように配置します。

### カバー率の自動算出（COUNTA + 数式）

各フェーズのカバー率は「5種別のうちURLが入っているセルの割合」で算出します。

**H2セル（認知フェーズのカバー率）:**

```excel
=COUNTA(B2:F2)/5

H5セル（比較フェーズのカバー率）:

=COUNTA(B5:F5)/5

H8セル（決定フェーズのカバー率）:

=COUNTA(B8:F8)/5

セルの書式を「パーセンテージ（小数0桁）」に設定してください。

条件付き書式でカバー率をハイライト

H2・H5・H8のカバー率セルに以下の条件付き書式を設定します。

条件 書式 意味
<40% 赤背景・白文字 危険：コンテンツ大幅不足
>=40% かつ <80% 黄背景・黒文字 注意：改善余地あり
>=80% 緑背景・白文字 良好：概ねカバーできている

設定手順（Excel）:

  1. H2:H8（カバー率セル）を選択
  2. ホーム → 条件付き書式 → 新しいルール
  3. 「数式を使用して…」を選び =H2<0.4 → 赤背景を設定
  4. 同様に =AND(H2>=0.4,H2<0.8) → 黄背景、=H2>=0.8 → 緑背景

②ダッシュボードシートの数式

フェーズ別PV合計

=SUM(戦略マップ!B3:F3)   ' 認知フェーズのPV合計
=SUM(戦略マップ!B6:F6)   ' 比較フェーズのPV合計
=SUM(戦略マップ!B9:F9)   ' 決定フェーズのPV合計

フェーズ別CV合計

=SUM(戦略マップ!B4:F4)   ' 認知フェーズのCV合計
=SUM(戦略マップ!B7:F7)   ' 比較フェーズのCV合計
=SUM(戦略マップ!B10:F10) ' 決定フェーズのCV合計

優先制作セルの自動マーキング

URLが空欄（＝コンテンツ未制作）のセルを自動で特定し、リスト化します。

戦略マップシート側の各URLセル（B2〜F8のURL行）に補助列を追加:

' I2セル（B2が空かどうかを判定）
=IF(B2="","★ "&$A2&"×"&B$1&" — 未制作","")

この数式をURL行すべてにコピーすると、未制作セルだけに「★ 認知×動画 — 未制作」のようなラベルが自動表示されます。

ダッシュボードに未制作リストを集約:

```excel ' ダッシュボードB2セル以降に、FILTER関数で未制作だけ抽出 =FILTER(戦略マップ!I2:I10, 戦略マップ!I2:I10<>"", "すべてカバー済み")
