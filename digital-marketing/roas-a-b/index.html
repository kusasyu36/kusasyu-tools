動画広告ROAS管理＆A_Bテスト計算シート

SNS動画広告AIクリエイティブ設計エンジン — 商品情報だけでSora2対応の脚本・絵コンテ・コピーを一括生成

動画広告ROAS管理＆A/Bテスト計算シート｜Excel完全ガイド

このシートでできること

  • 3クリエイティブ×4プラットフォーム＝12パターンを1シートで比較
  • 広告費・imp・click・CVを入力するだけでROAS・CPA・CTR・CVRを自動算出
  • EC/D2C業界ベンチマークとの比較が一目でわかる
  • 条件付き書式で「勝ちクリエイティブ」を自動ハイライト
  • Excel／Google Sheets両対応の数式設計

シート構成

A. ヘッダー行（1〜2行目）

列 項目 備考
A プラットフォーム YouTube / Meta / TikTok / LINE
B クリエイティブ名 A・B・C の3パターン
C 広告費（円） 手入力
D インプレッション 手入力
E クリック数 手入力
F CV数 手入力
G 売上（円） 手入力
H CTR（%） 自動計算
I CVR（%） 自動計算
J CPA（円） 自動計算
K ROAS（%） 自動計算
L 業界CTR ベンチマーク参照値
M 業界CVR ベンチマーク参照値
N 業界CPA ベンチマーク参照値
O 業界ROAS ベンチマーク参照値
P 判定 勝ち／普通／負け

B. データ行（3〜14行目：12パターン）

行 プラットフォーム クリエイティブ
3〜5 YouTube A・B・C
6〜8 Meta（Instagram/Facebook） A・B・C
9〜11 TikTok A・B・C
12〜14 LINE A・B・C

自動計算の数式一覧

すべて3行目の例。14行目までオートフィルでコピー。

基本KPI（H〜K列）

H3（CTR%）  = IF(D3=0, 0, E3/D3*100)
I3（CVR%）  = IF(E3=0, 0, F3/E3*100)
J3（CPA）   = IF(F3=0, 0, C3/F3)
K3（ROAS%） = IF(C3=0, 0, G3/C3*100)

勝ちクリエイティブ判定（P列）

同一プラットフォーム内でROASが最大のクリエイティブを「勝ち」と判定する。

P3 = IF(K3=MAX(OFFSET(K3, -MOD(ROW(K3)-3,3), 0, 3, 1)),
      IF(AND(K3>=O3, H3>=L3), "◎ 勝ち", "○ 普通"),
      IF(K3<O3*0.7, "× 負け", "△ 要改善"))

判定ロジック：

判定 条件
◎ 勝ち 同プラットフォーム内ROAS最大 かつ 業界ROAS・CTR以上
○ 普通 同プラットフォーム内ROAS最大だが業界基準未達
△ 要改善 ROAS最大ではないが業界の70%以上
× 負け 業界ROASの70%未満

Google Sheets互換メモ: OFFSET関数はGoogle Sheetsでも動作する。MOD(ROW())による相対参照もそのまま使用可能。

EC/D2C業界ベンチマーク参照値（L〜O列）

2025〜2026年の業界平均データ。プラットフォーム行ごとに固定値として入力する。

プラットフォーム CTR（%） CVR（%） CPA（円） ROAS（%）
YouTube 0.50 2.0 4,500 350
Meta 1.20 3.5 3,200 450
TikTok 1.80 2.8 2,800 380
LINE 0.80 4.0 3,800 400

補足: 上記は動画広告に限定したEC/D2C業界の中央値。商材単価や利益率により大きく変動するため、自社データが蓄積されたら上書き推奨。

条件付き書式の設定

設定1：ROAS列（K3:K14）のカラースケール

条件 書式
ROAS 500%以上 緑背景（#C6EFCE）＋太字
ROAS 300〜499% 黄背景（#FFEB9C）
ROAS 300%未満 赤背景（#FFC7CE）

Excel設定手順： 1. K3:K14を選択 2. ホーム → 条件付き書式 → 新しいルール 3. 「数式を使用して…」を選択

緑: =$K3>=500
黄: =AND($K3>=300, $K3<500)
赤: =$K3<300

設定2：判定列（P3:P14）のアイコン表示

判定テキスト 書式
◎ 勝ち 緑背景＋太字
× 負け 赤背景＋赤文字 
緑: =LEFT($P3,1)="◎"
赤: =LEFT($P3,1)="×"

設定3：業界比較ハイライト（H〜K列 vs L〜O列）

自社値が業界値を上回るセルを緑文字にする。

H列: =$H3>=$L3
I列: =$I3>=$M3
J列: =$J3<=$N3   ← CPAは低い方が良いので不等号逆
K列: =$K3>=$O3

集計エリア（16〜20行目）

シート下部にプラットフォーム別サマリーを設置する。

プラットフォーム別合計・平均

C16（YouTube広告費合計）  = SUM(C3:C5)
H16（YouTube平均CTR）     = AVERAGEIF(H3:H5,">"&0)
K16（YouTube平均ROAS）    = IF(SUM(C3:C5)=0, 0, SUM(G3:G5)/SUM(C3:C5)*100)

ROAS平均の注意: 単純平均ではなく加重平均（総売上÷総広告費）を使う。広告費の大きいクリエイティブの影響を正しく反映するため。

全体サマリー（20行目）

C20（総広告費）   = SUM(C3:C14)
K20（総合ROAS）   = IF(C20=0, 0, SUM(G3:G14)/C20*100)
J20（全体CPA）    = IF(SUM(F3:F14)=0, 0, C20/SUM(F3:F14))
P20（ベスト）     = INDEX(B3:B14, MATCH(MAX(K3:K14), K3:K14, 0))

使い方（3ステップ）

Step 1｜データ入力 C〜G列（広告費・imp・click・CV・売上）に各プラットフォームの管理画面から数値を転記する。週次または月次で更新。

Step 2｜自動判定を確認 H〜K列のKPIとP列の判定が自動で算出される。緑ハイライトのクリエイティブが現時点の勝者。

Step 3｜アクションを決定

判定結果 推奨アクション
◎ 勝ち 予算を集中投下。類似バリエーションを追加テスト
○ 普通 CTRまたはCVRのどちらがボトルネックか特定して改善
△ 要改善 サムネイル・冒頭3秒・CTAを変更して再テスト
× 負け 配信停止。予算を勝ちクリエイティブへ再配分

A/Bテスト運用のポイント

  1. 同時期・同予算で比較する — 時期ズレは季節要因が混入する
  2. 最低1,000クリック到達まで判定しない — サンプル不足の早期判断は誤判定の元
  3. 変数は1つずつ変える — サムネ＋CTA＋尺を同時に変えると勝因が特定できない
  4. 2週間サイクルで回す — テスト→判定→差替えを月2回ペースで継続

Google Sheetsで使う場合の補足

項目 対応
条件付き書式 表示形式 → 条件付き書式 → カスタム数式。Excel版と同じ数式がそのまま使える
OFFSET関数 Google Sheetsでも完全互換
色コード Excelのテーマカラーは自動変換されないため、カスタム色（#C6EFCE等）を手動設定
共同編集 シート保護（データ → シートを保護）で数式セルをロックし、入力セルのみ編集可にする

このテンプレートを使えば、動画広告の勝ちパターンをデータで即判断できます。感覚的な「なんとなく良さそう」から脱却し、ROAS起点の意思決定を始めましょう。
