スマホ1台で撮れる！ショート動画撮影チェックリスト

使い方: 撮影の前・中・後の3ステップでチェックを入れるだけ。「ショート動画シナリオ＆SNS投稿ジェネレーター」で生成したシナリオを手元に用意し、このリストを見ながら撮影すれば30秒動画が完成します。印刷してA4・1枚で使えます。スマホからはスクリーンショット保存が便利です。

STEP 1｜撮影前の準備（所要5〜10分）

1-1. シナリオ確認

  • [ ] 生成したシナリオを印刷 or スマホ表示した → [記入例: 「新メニュー紹介」シナリオをスマホのメモアプリに貼り付け済み]
  • [ ] セリフを3回声に出して読んだ → [記入例: 30秒に収まるか確認。早口になる場合はセリフを1文削る]
  • [ ] カット割りの順番をメモした → [記入例: ①引きの全体ショット→②寄りの商品アップ→③顔出しでひと言]

1-2. 照明

  • [ ] 撮影場所の明るさを確認した → [記入例: 窓の近くに移動し、自然光が顔に当たる位置をキープ]
  • [ ] 逆光になっていない → [記入例: 窓を背にせず、窓に向かって立つ/座る]
  • [ ] 照明を補助で用意した（必要な場合） → [記入例: デスクライトを斜め45度上から当てる / 100均のLEDリングライト使用]

1-3. 背景

  • [ ] 背景に余計なものが映っていない → [記入例: 段ボール・私物・散らかった棚を片付けた]
  • [ ] ブランドが伝わる要素がある → [記入例: 店名ロゴ入りのれん / 施術イスが見える位置にセット]
  • [ ] 背景の色味が商品と被らない → [記入例: 白い商品なら背景は木目やグレーの壁を選ぶ]

1-4. 音声

  • [ ] 周囲の雑音をチェックした → [記入例: エアコン・換気扇の音が入らない場所へ移動]
  • [ ] マイクテスト10秒録音→再生した → [記入例: スマホ内蔵マイクで十分聞こえるか確認。声が小さければピンマイク使用]
  • [ ] BGMを後付けするか決めた → [記入例: シナリオのBGM指示を確認し、編集時にCapCutで追加予定]

1-5. スマホ設定

  • [ ] 通知をオフにした（おやすみモード） → [記入例: 設定→集中モード→ONにして着信音・通知を遮断]
  • [ ] ストレージ空き容量を確認した → [記入例: 30秒動画×5テイクで約500MB。最低1GB空きを確保]
  • [ ] 画質を設定した → [記入例: 1080p/30fps（設定→カメラ→ビデオ撮影）]
  • [ ] スマホを横置き or 縦置きで固定した → [記入例: ショート動画は縦置き（9:16）。100均ミニ三脚+スマホホルダーで固定]

業種別メモ - 🍽 飲食: 料理は自然光が最重要。蛍光灯だけだと青白く見えるので窓際テーブルを撮影スポットにする - 💇 美容: 施術イスやミラーの映り込みに注意。鏡にスマホが映らない角度を先に確認 - ⚖ 士業: 個人情報が映った書類・PC画面が背景に入らないよう必ずチェック

STEP 2｜撮影中のコツ（所要5〜15分）

2-1. カメラ位置・アングル

  • [ ] カメラは目線の高さにセットした → [記入例: テーブルに三脚を置き、立った状態で目とレンズが同じ高さ]
  • [ ] 被写体がフレームの中央〜やや上にいる → [記入例: 頭上に拳1つ分の余白を空ける]
  • [ ] 寄り（アップ）と引き（全体）の切替を意識した → [記入例: シナリオの「カット2: 商品アップ」で手持ちでグッと寄る]

2-2. 話し方

  • [ ] 最初の3秒で「結論 or 問いかけ」を言った → [記入例: 「この1品で売上が月5万円変わりました」/「知ってましたか？」]
  • [ ] 普段の1.2倍の声量で話した → [記入例: 胸に手を当てて声を出す感覚。ボソボソはNG]
  • [ ] 表情は「口角を上げた状態」をキープした → [記入例: 撮影前に「ウィスキー」と3回言って口角をほぐす]
  • [ ] 1テイク30秒で収まった → [記入例: スマホのタイマーで計測。35秒以上なら後半のセリフを1文カット]

2-3. テイク管理

  • [ ] 最低3テイク撮影した → [記入例: テイク1=練習、テイク2=本番、テイク3=予備の感覚]
  • [ ] 各テイクの冒頭に指パッチンで音を入れた → [記入例: 編集で頭出しが楽になる。手を叩くでもOK]
  • [ ] 撮影後すぐにプレビュー確認した → [記入例: 音声が入っているか・ブレていないか・背景に問題がないかをその場でチェック]

業種別メモ - 🍽 飲食: 湯気・とろける瞬間など「シズル感」は1テイク勝負。料理が冷める前にアップショットを最優先で撮る - 💇 美容: ビフォーアフターは同じアングル・同じ照明で撮ること。比較が伝わらないと効果ゼロ - ⚖ 士業: カメラ目線でゆっくり話すと信頼感UP。手元の資料を見ながら話すとテレビの解説風になり好印象

STEP 3｜撮影後の仕上げ（所要10〜20分）

3-1. 編集ポイント

  • [ ] 無料アプリを用意した → [記入例: CapCut（無料・商用可）をインストール済み]
  • [ ] 冒頭の無音・余白をカットした → [記入例: 開始0.5秒以内に声 or 動きが始まるようトリミング]
  • [ ] テロップ（字幕）を入れた → [記入例: セリフをそのままテロップに。文字サイズはスマホで読める大きさ（画面幅の70%以内）]
  • [ ] シナリオ指示通りのBGMを設定した → [記入例: CapCut内蔵の「明るいポップ」系BGMを音量20%で追加]
  • [ ] 最後に行動喚起（CTA）テロップを入れた → [記入例: 「フォローで最新情報をGET→」「プロフのリンクから予約」]

3-2. 書き出し・最終チェック

  • [ ] 書き出し設定は1080×1920（縦9:16）にした → [記入例: CapCut→エクスポート→1080p]
  • [ ] 完成動画を通しで再生して確認した → [記入例: 音ズレ・テロップ誤字・画面の暗い箇所がないか最終チェック]
  • [ ] 動画の長さが15〜30秒に収まっている → [記入例: 28秒。60秒を超えるとリーチが落ちるので厳守]

3-3. 投稿の最適化

  • [ ] 投稿文をジェネレーターで生成済み → [記入例: 「ショート動画シナリオ＆SNS投稿ジェネレーター」で同テーマのX投稿文5本+Instagram投稿文5本を生成]
  • [ ] ハッシュタグを5〜10個つけた → [記入例: #業種名 #地域名 #サービス名 #TikTok #Reels]
  • [ ] 投稿時間帯を確認した → [記入例: 下記の表を参照]
プラットフォーム ベスト投稿時間 理由
TikTok 19:00〜22:00 帰宅後のスマホ視聴ピーク
Instagram Reels 12:00〜13:00 / 20:00〜21:00 昼休み＋夜のリラックスタイム
X（旧Twitter） 7:00〜8:00 / 12:00〜13:00 通勤・昼休みのタイムライン閲覧
  • [ ] 投稿予約をセットした → [記入例: Instagramのプロフェッショナルダッシュボードで翌日20:00に予約]

業種別メモ - 🍽 飲食: 投稿は火〜木が反応◎。ランチ動画は11:00投稿、ディナーは17:00投稿が効果的 - 💇 美容: ビフォーアフター動画は保存率が高い。「保存してね」のCTAを必ず入れる - ⚖ 士業: 顔出し動画は「先生」感を出すため、スーツ＋落ち着いた背景で統一。週1投稿で十分効果あり
