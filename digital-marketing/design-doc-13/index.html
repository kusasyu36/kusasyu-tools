1. サービス概要

本サービスは、GoogleのE-E-A-T（Experience・Expertise・Authoritativeness・Trustworthiness）評価基準に準拠したSEOコンテンツを、AIと専門家のハイブリッド体制で制作するパイプラインである。

E-E-A-T 4要素の担保方法

要素 定義 担保方法
Experience（経験） 実体験に基づく情報 専門家インタビュー・事例取材を記事に反映。一次情報を必ず1つ以上含める
Expertise（専門性） 分野の深い知識 該当業界の有資格者または実務経験5年以上の監修者をアサイン。著者プロフィールを記事に明記
Authoritativeness（権威性） 情報源としての信頼 公的機関・学術論文・業界団体の一次ソースを引用。被リンク獲得を意識した構成設計
Trustworthiness（信頼性） 正確性・透明性 ファクトチェック工程を必須化。引用元URL明記、最終更新日の表示、運営者情報の整備を支援

2. 制作パイプライン（5ステップ）

Step 1：キーワード分析（1〜2営業日）

  • 検索ボリューム・競合難易度・検索意図を3軸で評価
  • ツール：Ahrefs / Search Console / ラッコキーワード
  • 成果物：キーワードマップ（主軸KW＋関連KW＋ロングテール）
  • クライアント確認：対象KWの優先順位を合意

Step 2：構成案作成（1〜2営業日）

  • 検索上位10記事のh2/h3構成を分析し、網羅性と差別化ポイントを設計
  • E-E-A-T要素の挿入箇所を構成段階で指定（体験談の位置、引用ソース候補など）
  • 成果物：見出し構成＋各セクション200字以内の要旨メモ
  • クライアント確認：構成承認後に執筆へ進行

Step 3：AIドラフト執筆（1営業日）

  • Claude / GPT-4を活用し、構成案に基づく初稿を生成
  • AI出力のまま納品しない。あくまで「素材」として扱う
  • この段階でのE-E-A-Tスコア目標：60点以上／100点

Step 4：専門家監修（2〜3営業日）

  • 該当分野の監修者が事実確認・加筆修正を実施
  • 監修者の実名・資格・経歴を記事末尾に掲載
  • 一次情報（体験・独自データ・専門的見解）を追記
  • この段階でのE-E-A-Tスコア目標：80点以上／100点

Step 5：SEO最適化・公開準備（1営業日）

  • タイトル・メタディスクリプション・内部リンク・構造化データの最適化
  • Core Web Vitals対応の画像圧縮・alt設定
  • 最終E-E-A-Tスコアチェック → 80点未満は差し戻し
  • 成果物：WordPress等CMSへの入稿データ一式

1記事あたり標準リードタイム：5〜8営業日

4. 品質基準：E-E-A-Tスコアリング

採点方法（100点満点）

評価軸 配点 主な評価項目
Experience 25点 一次情報の有無、体験ベースの記述量、具体的事例数
Expertise 25点 監修者の資格・経歴、専門用語の正確性、情報の深さ
Authoritativeness 25点 引用ソースの質と数、著者情報の充実度、外部評価
Trustworthiness 25点 ファクトチェック通過率、更新日明記、透明性（利益相反開示等）

合格ライン

基準 スコア
最低合格ライン 80点以上（全記事必須）
各要素の最低点 15点以上／25点（1要素でも下回れば差し戻し）
目標水準 85点以上

差し戻しフロー

  1. 80点未満 → 監修者へ再修正依頼（追加費用なし）
  2. 再修正後も75点未満 → 別の監修者をアサインし再執筆
  3. 月次レポートで全記事の平均スコアを報告
