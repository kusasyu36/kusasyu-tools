インハウスマーケ移行 90日実行ロードマップ

BtoBマーケ戦略コーチ＆KPIシミュレーター 連動ガイド｜対象：年商1〜10億円 BtoB中小企業

全体進捗: 0 / 36 タスク完了

Phase 1：土台づくり（1〜30日目）

Week 1｜現状の棚卸し

Week 2｜ターゲット・KPI設定

Week 3｜Webサイト最低限整備

Week 4｜初回コンテンツ公開

✓ GO基準（Phase 2に進む条件）
①GA4でCV計測が動作 ②KPI目標が数値で合意済み ③記事1本＋CTA設置完了
→ 3つ全て満たせばPhase 2へ。1つでも未達ならWeek 4を延長して対応。
⚠ よくあるつまずき
・「完璧な記事」を目指して公開できない → 60点で公開、後から改善がBtoBの鉄則
・GA4の設定がわからない → Google公式ヘルプの「測定IDの設置」だけやれば十分
・KPIが大きすぎる → 月間リード5件から開始。過去実績がなければ小さく始める

Phase 2：施策の実行と検証（31〜60日目）

Week 5｜コンテンツ量産体制

Week 6｜リード獲得の仕組み化

Week 7｜広告テスト（小額）

Week 8｜データ振り返り

✓ GO基準（Phase 3に進む条件）
①月間リード3件以上を獲得 ②CPA 15,000円以下（広告実施の場合） ③記事4本以上公開済み
→ 2つ以上満たせばPhase 3へ。全て未達の場合、施策の見直し（ターゲット・チャネル変更）を実施。
⚠ よくあるつまずき
・広告費が怖くて出稿できない → 日予算1,000円で十分。データが取れることが最優先
・リードが来てもフォローしない → 48時間以内の架電ルールを営業と合意しておく
・数値を見ない → 毎週金曜15分だけ「先週の数字」を見る習慣をセットする

Phase 3：自走体制の確立（61〜90日目）

Week 9｜改善サイクルの定着

Week 10｜ナーチャリング開始

Week 11｜業務フロー標準化

Week 12｜次の90日計画

✓ 90日完了基準
①代理店費の月額5〜30万円のうち自社代替できた割合を算出 ②月間リード目標の達成率50%以上 ③運用手順書が完成し、担当者以外でも作業可能
→ 全て満たせばインハウス化の基盤完成。未達項目は次の90日で重点的に改善。
⚠ よくあるつまずき
・「全部自社でやる」に固執 → 専門性が高い領域（SEO技術監査等）は外注が効率的
・成果を経営者に報告しない → 数字で見せないと予算・人員の確保ができない
・手順書を作らない → 属人化して担当者が変わった瞬間に止まる
