1. ダッシュボード概要

1.1 目的

マーケティング専任担当者を持たない中小BtoB企業が、専門知識なしに「次に何をすべきか」を判断できる分析基盤を提供する。GA4・Google Search Consoleのデータを自動取得・統合し、コンテンツマーケティングのPDCAサイクルを数値で回せる状態を実現する。

1.2 設計原則

原則 説明
指標は最小限 BtoB企業に必要な指標だけに絞る。PV至上主義を排し、リード獲得貢献度を最上位KPIに据える
解釈不要 数値の羅列ではなく、自然言語による「診断コメント」を自動生成する
アクション直結 すべてのデータ表示に「推奨アクション」を紐づける
非マーケター向けUI 経営者・営業部長が週1回5分で確認できる情報密度に制御する

1.3 データソースと取得方式

データソース 取得API 更新頻度 主要取得項目
GA4 Google Analytics Data API v1 日次（AM 6:00 JST） セッション、エンゲージメント率、コンバージョン、流入経路、ページ別指標
Search Console Search Console API 日次（AM 6:00 JST） クエリ別表示回数・クリック数・CTR・平均掲載順位
内部DB 自社API リアルタイム 記事メタ情報、公開日、ターゲットKW、カテゴリ、制作コスト

認証フロー: OAuth 2.0によるワンクリック連携。初回セットアップウィザードでGA4プロパティID・Search Consoleサイトを選択するだけで接続完了。

2. KPI体系設計

2.1 KPIツリー

[最上位] コンテンツ経由リード獲得数（MQL）
    │
    ├── [中間] コンバージョン率（CVR）
    │       ├── CTA クリック率
    │       ├── フォーム到達率
    │       └── フォーム完了率
    │
    ├── [中間] 有効トラフィック量
    │       ├── オーガニック検索セッション数
    │       ├── ターゲットKW TOP10率
    │       └── 指名検索比率
    │
    └── [中間] コンテンツ品質スコア
            ├── 平均エンゲージメント時間
            ├── 直帰率（逆指標）
            ├── 回遊率（セッションあたりPV）
            └── SNSシェア数

2.2 KPI定義一覧

KPI名 定義 算出式 計測単位 目標基準値
コンテンツ経由MQL数 コンテンツページを経由してCV（資料DL・問い合わせ）に至ったユニークユーザー数 GA4イベント generate_lead のうち page_referrer がブログ記事のもの 件/月 業界別に初期設定（製造業: 5件、IT/SaaS: 15件、士業: 8件）
オーガニックCVR オーガニック流入セッションのうちCVに至った割合 CV数 ÷ オーガニックセッション数 × 100 % 1.0〜3.0%（BtoB平均）
ターゲットKW TOP10率 設定したターゲットKWのうちGoogle検索TOP10に入っている割合 TOP10入りKW数 ÷ 全ターゲットKW数 × 100 % 6ヶ月後に30%以上
平均エンゲージメント時間 ユーザーがページをアクティブに閲覧していた平均時間 GA4 averageSessionDuration（エンゲージセッション） 秒 120秒以上
コンテンツROI コンテンツ制作投資に対するリード獲得の費用対効果 (MQL数 × 平均リード単価) ÷ 月間コンテンツ制作コスト 倍率 3.0x以上（6ヶ月後）
記事品質スコア エンゲージメント・SEO・CV貢献の複合評価 後述の独自スコアリング（100点満点） 点 70点以上

2.3 記事品質スコア算出ロジック

記事品質スコア（100点満点）=
    エンゲージメント指標（40点）
        平均滞在時間スコア    : 15点（120秒以上で満点、線形補間）
        直帰率スコア          : 10点（40%以下で満点、線形補間）
        回遊率スコア          :  8点（1.5PV/session以上で満点）
        スクロール深度スコア  :  7点（75%以上で満点）

    SEOパフォーマンス指標（35点）
        ターゲットKW順位スコア: 15点（1位=15, 2位=14, ... 10位=6, 圏外=0）
        表示回数成長率スコア  : 10点（前月比+20%以上で満点）
        CTRスコア             : 10点（業界平均CTR比で算出）

    CV貢献指標（25点）
        直接CV貢献スコア      : 15点（CV発生記事は15点、なければ0点）
        アシストCV貢献スコア  : 10点（CVパス上に存在する場合、関与度で按分）

■ 問題分析
  1. 滞在時間 48秒 → コンテンツ深度不足
  2. 直帰率 78% → 検索意図とのミスマッチの可能性
  3. 順位が15位→22位に下落 → 競合記事の品質向上

■ 競合上位記事との差分（自動分析）
  - 上位記事の平均文字数: 4,200字（当記事: 1,800字）
  - 上位記事にあって当記事にない見出し:
    ・材料費削減の具体的手法（3記事/5記事が言及）
    ・自動化による工数削減事例（4記事/5記事が言及）
    ・コスト削減のROI試算方法（3記事/5記事が言及）

■ 推奨リライト内容
  1. 以下の見出しを追加:
     H2: 材料費削減の3つの実践手法
     H2: 成形工程の自動化による工数削減事例
     H2: コスト削減効果のROI計算テンプレート
  2. 文字数を3,500字以上に拡充

  4. CTA「コスト削減診断シート無料ダウンロード」を設置

CTR Click Through Rate。表示回数に対するクリック数の割合
エンゲージメント時間 GA4固有指標。ユーザーがページをアクティブに閲覧していた時間
品質スコア 本システム独自指標。エンゲージメント・SEO・CV貢献の3軸で算出する100点満点の複合指標

アシストCV 最終CVページではないが、CVに至るユーザージャーニー上で閲覧された記事による間接的なCV貢献
RLS Row Level Security。データベースレベルでテナント間のデータを分離する仕組み
